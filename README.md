# stitchNapoliPizzaElite

Modern React 18 + TypeScript + NPM architecture using:

- **Feature-driven modules** (`src/features/*`)
- **App shell** (`src/app/*`) with routes, providers and guards
- **Shared library** (`src/shared/*`) for reusable pieces
- **React Router + lazy loading + Suspense**
- **TanStack Query** for server-state orchestration
- **Zustand** for local application state slices
- **Custom hooks and service layers** (Repository + Manager)

## Scripts

```bash
npm install
npm run dev
npm run build
npm run lint
```
